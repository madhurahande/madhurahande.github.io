Communication Architecture

Project information

  • Category: Patient Monitor NX5
  • Year: 2024 - 2025

Designed and implemented the communication framework using UART, SPI protocols for real-time data transfer between multiple sensor modules (ECG, NIBP, Respiration, Temperature).

Developed UART protocol for high-speed, low-latency communication between microcontroller and the main processor.

Ensured secure and synchronized data transmission, reducing packet loss and enhancing system reliability.
