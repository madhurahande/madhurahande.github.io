# Portfolio — Madhura Hande Deshmukh

Modern Astro portfolio for [madhurahande.github.io](https://madhurahande.github.io).

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:4321](http://localhost:4321).

## Before deploying

1. **Add your resume PDF** at `public/resume.pdf` (linked from the hero and navigation).
2. In GitHub repo **Settings → Pages**, set source to **GitHub Actions**.
3. Merge `astro-rebuild` into `main` and push to deploy.

## Content

- Edit site metadata in `src/data/site.ts`
- Edit projects in `src/data/projects.ts`
- Edit experience and education in `src/data/experience.ts`
- Contact form uses Formspree (`xknajvjn`)

The previous iPortfolio HTML site is archived in `legacy/`.
